ATRIBUTOS EN HTML


Todos los elementos HTML pueden tener atributos

Los atributos proporcionan información adicional sobre los elementos

Los atributos siempre se especifican en la etiqueta de inicio

Los atributos generalmente vienen en pares de nombre/valor como: name="value"

Existen diversos tipos de atriutos, tales como:

El atributo (href), que especifica la dirección URL de la pagina a la que va el vínculo

El atributo (src), que especifica la ruta de acceso a la imagen que se va a mostrar

El atributo (alto y ancho) que especifican el ancho y alto de la imagen en pixeles

El atributo (alt) requerido para la etiqueta especifica un texto alternativo para una imagen, si la imagen por alguna razón no se puede mostrar

El atributo de estilo, que se usa para añadir estilos a un elemento, como color, fuente, tamaño y mas

Y el atributo lang, el cual se utiliza para declarar el idioma de la página web


Ejemplos de los atributos anteriores serian:

(href) --> (Visit W3Schools)

(src) --> ()

(alto y ancho) --> ()

(alt) --> ()

(de estilo) --> (style="color:red;">This is a red paragraph.)

(lang) --> (lang="en")
