Enlaces en HTML

Hipervínculos

Los enlaces en HTML son hipervínculos. Al clicar en un enlace, este te lleva a otro documento.

Al mover el cursor sobre el enlace este se convierte en una pequeña mano.

Nota: Un enlace no tiene por qué ser texto, puede ser una imagen o cualquier otro elemento de HTML!

Sintaxis

En HTML la etiqueta para un hipervínculo se define con <a>. Tiene la siguiente sintaxis:

<a href="url">texto del hipervínculo</a>

La parte más importante del elemento <a> es el atributo href pues indica el destino del hipervínculo.

El texto del hipervínculo es la parte que será visible para el lector.

Clicar en el texto mandará al lector a la dirección URL especificada. El siguiente ejemplo es un hipervínculo a la direccion edib.com

<a href="https://www.edib.com/">Visita Edib</a>

Por defecto, los hipervínculos apareceran de esta forma en todos los navegadores:

  • Un hipervínculo no visitado estará subrayado y de color azul.
  • Un hipervínculo visitado estará subrayado y de color morado.
  • Un hipervínculo activo estará subrayado y de color rojo.

Consejo: Los hipervínculos pueden tener estilos de CSS para tener otra apariencia.

El atributo target

Por defecto, la página a la que se direcciona se abrirá en la misma ventana/pestaña que se esté utilizando. Para cambiar esto se debe especificar otro target para el vínculo.

Puede tener uno de estos diferentes valores:

  • _self - Por defecto. Abre el documento en la misma ventana o pestaña.
  • _blank - Abre el documento en otra ventana o pestaña.
  • _parent - Abre el documento en el marco padre.
  • _top - Abre el documento en el cuerpo entero de la ventana.

Ejemplo:

<a href="https://www.edib.com/" target="_blank">Visita Edib</a>

URL Absolutas vs URL Relativas

Los ejemplos anteriores usan una URL absoluta (una dirección web completa) en el atributo href.

Un vínculo local (un vínculo a una página dentro del mismo sitio web) se especifica con una URL relativa (sin la parte "https://wwww."):

URL Absoluta:

<a href="https://www.edib.com/">Visita Edib</a>


URL Relativa:

<a href="parrafos.html">Sección de párrafos</a>

Uso de imágenes como vínculos

Para usar una imagen como vínculo, simplemente escribe la etiqueta <img> dentro de la etiqueta <a>:

<a href="ejemplo.html">

<img src="smiley.gif" alt="Tutorial de HTML" style="width:42px;height:42px;">

</a>

Vínculo a una dirección email

Dentro del atributo href se puede utilizar mailto: para crear un vínculo que abre el programa de email del usuario (y permite mandar un nuevo email):

<a href="mailto:alguien@ejemplo.com">Enviar email</a>

Botones como vínculos

Para utilizar botones de HTML como vínculo tenemos que añadir algo de código JavaScript.

JavaScript nos permite especificar qué ocurre con ciertos eventos, como por ejemplo al clicar un botón:

<button onclick="document.location='default.asp'">Tutorial HTML<&button>
