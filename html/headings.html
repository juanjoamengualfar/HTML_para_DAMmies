Cabeceras de HTML

Las cabezeras son elementos que dan formato a los títulos o subtítulos de una página web.

Escribir una cabecera

Las cabeceras se definen con las etiquetas desde <h1> hasta <h6>.

El número representa el título que se formará, siendo <h1> el más grande y <h6> el más pequeño.

Por ejemplo:

Esto está escrito con <h1>

Esto está escrito con <h2>

Esto está escrito con <h3>

Esto está escrito con <h4>

Esto está escrito con <h5>
Esto está escrito con <h6>

Atención: observa que en las cabeceras se ha creado un pequeño margen con respecto al texto normal

Usos de las cabeceras

Las cabeceras añaden estructura a la página web.

También son importantes para indexar el contenido de la propia página.

Si se observa este tutorial, observamos que <h1> ha sido usado para el título y <h2> son divisores de contenido.

Atención: las cabeceras no deberían ser utilizadas para crear texto grande o en negrita.

Cambiar el tamaño de las cabeceras

Al igual que ocurre con el texto, puedes modificar las cabeceras usando CSS, ya sea exerno, interno o usando el atributo style en la etiqueta de inicio.
